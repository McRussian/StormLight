# StormLight — Штормсердце

Настольная игра по вселенной «Архив Буресвета» Брэндона Сандерсона.

**Жанр:** гонка / управление ресурсами / тактическое блокирование  
**Игроки:** 2–4 · **Раундов:** 3–4 · **Статус:** концепт, механика зафиксирована

---

## Об игре

Игроки собирают гемсердца на гексагональной карте Равнин Пепла до удара шторма.
Карта генерируется в каждой партии: тайлы выкладываются по правилу стыковки ущелий — они образуют непрерывные линии через всю карту. Мост позволяет пересечь ущелье в обход карты движения, но остаётся лежать и может быть украден соперником.

→ [Правила и механика](doc/stormlight_plains.md)  
→ [План симуляции и подбора параметров](doc/stormlight_simulation.md)

---

## Цель репозитория

Python-симуляция для балансировки игровых параметров:

1. **Генератор карты** — ядро + случайные тайлы по правилу стыковки, метрики баланса
2. **Симуляция ботов** — жадный бот, затем несколько стратегий
3. **Прогон 1000+ партий** — подбор длины раунда, радиуса карты, состава колоды движения

## Запуск

```bash
pip install -r requirements.txt
python generate_map.py
python generate_map.py --radius 5 --seed 42 --output my_map.png
```
